How To...

Browse PixelCons
Search by Name
At the top of the the Browse page, there is a search bar to search for PixelCons by name. The results will be sorted by closest match first with a page select located at the bottom. Leaving this blank simply fetches all PixelCons currently in existence.
Search PixelCons for Sale
You can search for PixelCons that are for sale on OpenSea. Any other market that supports ERC-721 NFTs should work too.
PixelCon Result Details
PixelCons are displayed with different indicators to illustrate their details at a glance. PixelCons that you own will have the account icon in the bottom right corner. If a PixelCon is part of a collection, it will have a collection icon in the bottom left corner. Additionally, hovering your cursor over each result will show specifics about what collection it might belong to.



View/Create PixelCons
View PixelCon
Clicking on a PixelCon from any page, will take you the PixelCon Details page. Here you can see additional information about a PixelCon like Name, Number, Date Created, Creator and Owner. You can also see details about the PixelCons Collection if applicable.
Edit PixelCon
On the PixelCon Details page, if you are the current owner you will see a menu button with a drop down for different actions. As the owner, you have the option of sending it to another account. Additionally as the creator and current owner, you can rename the PixelCon.
View Collection
Clicking on the collection icon from the PixelCon Details page, will take you the Collection Details page. Here you can see all the PixelCons in the collection as well as the collection creator.
Edit Collection
On the Collection Details page, if you are the owner and creator of all PixelCons in the collection you will see a menu button with a drop down for different actions. These actions include renaming and clearing the collection.
View Creator
Clicking on the creator icon from the PixelCon Details or Collection Details page, will take you the Creator Details page. Here you can see all the PixelCons that were created by the creator account.
Create PixelCon
On the Create page, you can draw your own PixelCons from a palette of sixteen colors. When you are finished, click the create button to check that it does not already exist. All PixelCons must be unique, so make something new and creative!



Your Account
Account Activity
When logged in, your account icon will appear in the top right corner of the page. Clicking on this icon will display any pending transactions you've made as well as links to view their status on Etherscan. A spinner will also appear on this icon if you currently have any transactions being processed on the Ethereum network.
View Wallet
On the Account page, you can view the PixelCons that you currently own. You can sort your collection by either Name or Date Created.
Sell PixelCon
You can list PixelCons for sale through OpenSea. Visit your OpenSea account page and make listings from there or manage existing ones and potential offers.
Send PixelCon
You can send ownership of your PixelCons to someone else by navigating to the PixelCon Details page. On this page you will see a menu button with a drop down of actions including one to send your PixelCon to another account.
Create Collection
At the top of the Account page, there is a toggle to switch to the creator view. From this view you can can create collections by clicking the create collection button and selecting the PixelCons you want to be grouped into a collection. You can only select PixelCons that you own and are not already in a collection.
