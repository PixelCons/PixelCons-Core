How To...

Browse PixelCons
Search by Name
At the top of the the Browse page, there is a search bar to search for PixelCons by name. The results will be sorted by closest match first with a page select located at the bottom. Leaving this blank simply fetches all PixelCons currently in existence.
Search PixelCons for Sale
You can search for PixelCons that are for sale on OpenSea. Any other market that supports ERC-721 NFTs should allow for buying/selling/trading as well.
PixelCon Result Details
PixelCons are displayed with different indicators to illustrate their details at a glance. PixelCons that you own will have the account icon in the bottom right corner. If a PixelCon is part of a collection, it will have a collection icon in the bottom left corner. Hovering your cursor over each result will show specifics about what collection it might belong to.



View Details
PixelCon Details
Clicking on a PixelCon from any page, will take you to the PixelCon Details page. Here you can see additional information about a PixelCon like Name, Number, Date Created, Creator and Owner. You can also see details about the PixelCons Collection if applicable.
Collection Details
Clicking on the collection icon from the PixelCon Details page, will take you the Collection Details page. Here you can see all the PixelCons in the collection as well as the collection details like Name, Number and Creator.
Creator Details
Clicking on the creator icon from the PixelCon Details or Collection Details page, will take you the Creator Details page. Here you can see all the PixelCons that were created by the creator account.



Create PixelCons
Create PixelCons
On the Create page, you can draw your own PixelCons from a palette of sixteen colors. All PixelCons must be unique, so make something new and creative! From the Advanced tab, you can create multiple PixelCons from an imported file.
Create Collection
On the Create page, you can organize your created PixelCons into Collections from the Collection tab. From this page you can select the PixelCons you want to be grouped into a collection. Only PixelCons you currently own, created and are not already in a collection are availabe to select.
Edit PixelCon
On the PixelCon Details page, if you are the current owner you will see a menu button with a drop down for different actions. If you are the current owner and creator, you can rename the PixelCon. If you are the owner, you can send the PixelCon to another account.
Edit Collection
On the Collection Details page, if you are the current owner and creator of all PixelCons in the collection you will see a menu button with a drop down for different actions. These actions include renaming and clearing the collection.



Your Account
Account Activity
When logged in, your account icon will appear in the top right corner of the page. Clicking on this icon will display any pending transactions you've made as well as links to view their status in a block explorer. A spinner will also appear on this icon if you currently have any transactions being processed by the network.
View Wallet
On the Account page, you can view the PixelCons that you currently own. You can sort your collection by either Name or Date Created.
Send PixelCons
On the Account page, there is a menu button with a drop down for different actions including sending you PixelCons to a different account. Select the PixelCon you wish to send, and click Send.
Sell PixelCons
You can list PixelCons for sale through OpenSea. Visit your OpenSea account page to make listings from there or manage existing ones and potential offers.
